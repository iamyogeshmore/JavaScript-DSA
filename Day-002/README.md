## 📝 JavaScript Questions

### 1. Swap Numbers
**File:** `Problem008-SwapNumbers.js`

**Concept:** Variable manipulation, arithmetic operations

**Problem Statement:** Swap two numbers with and without using a temporary variable.

**Expected Output:**
```
Input: a = 5, b = 10
Output: After swapping: a = 10, b = 5
```

### 2. Number Check
**File:** `Problem009-NumberCheck.js`

**Concept:** Conditional statements, ternary operator

**Problem Statement:** Check whether a number is positive, negative, or zero.

**Expected Output:**
```
Input: -5
Output: -5 is a Negative number.
```

### 3. Largest of Three
**File:** `Problem010-LargestOfThree.js`

**Concept:** Conditional statements, logical operators, ternary operator

**Problem Statement:** Find the largest among three numbers.

**Expected Output:**
```
Input: 10, 25, 15
Output: 25 is Largest.
```

### 4. Print 1 to N
**File:** `Problem011-Print1ToN.js`

**Concept:** Loops (for, while), iteration

**Problem Statement:** Print numbers from 1 to N.

**Expected Output:**
```
Input: N = 5
Output: 1 2 3 4 5
```
